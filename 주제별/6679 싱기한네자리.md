## 백준온라인저지 BOJ 6679: 싱기한 네자리 숫자

문제는 [여기](https://www.acmicpc.net/problem/6679)를 클릭해주세요.



### Source Code

```cpp
#include <iostream>
using namespace std;

int main(void) {
	for (int i = 1000; i < 10000; i++) {
		int dec =  i;	int duo = i;	int hexa = i;
		int a = 0;	int b = 0;	int c = 0;

		//decimal
		while(dec > 0){
			a += dec % 10;
			dec /= 10;
		}
		//duodecimal
		while (duo > 0) {
			b += duo % 12;
			duo /= 12;
		}
		//hexadecimal
		while (hexa > 0) {
			c += hexa % 16;
			hexa /= 16;
		}

		//result
		if (a == b && a == c) {
			cout << i << endl;
		}
	}
	return 0;
}
```
