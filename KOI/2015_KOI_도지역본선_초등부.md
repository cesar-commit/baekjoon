## 1. 10부제
문제를 보기 위해 [여기](https://www.acmicpc.net/problem/10797)를 클릭하세요.

```cpp
#include <stdio.h>

int a, b, num;
int main(void)
{
  //숫자가 같으면 위반
  scanf("%d", &a);
  for(int i=0;i<5;i++)
  {
    scanf("%d", &b);
    if(a==b)
      num++;
  }

  printf("%d", num);
  return 0;
}
```
